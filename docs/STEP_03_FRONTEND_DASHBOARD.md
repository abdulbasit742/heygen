# Step 03 — Frontend Dashboard

Added a Vite React dashboard for creating and tracking AI avatar video projects.

## Files Added
- `frontend/src/api.js`

## Files Updated
- `frontend/src/App.jsx`
- `frontend/src/style.css`

## Features
- Create video project form
- Title, prompt, language, tone, avatar, voice, and platform settings
- Backend API client
- Project progress polling
- Recent project list
- Responsive layout

## Run
```bash
cd backend && npm run dev
cd frontend && npm run dev
```
