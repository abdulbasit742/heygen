# Step 01 — Base Setup

This step adds the common foundation for the AI avatar video generator MVP.

## Added
- Root `.env.example`
- Shared JavaScript/JSDoc types
- Standard platform video sizes
- Safer app direction: creator video generation, not spam automation

## Commands

```bash
cd heygen_clone_starter
cp .env.example .env
npm install
npm run dev
```

## Notes
Use official APIs and user-owned media only. Do not clone real people, brands, or copyrighted avatars without permission.
